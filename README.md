# BedwarsItem

用途与[BedwarsScoreBoardAddon](https://github.com/LinMoyuu/BedwarsScoreBoardAddon)一致。

因此，本项目存在的意义是为了复刻《花雨庭》服务器的起床战争游戏体验。

当本插件与配套插件完成时，笔者会释出示例端作为参考。

本项目代码基本来自[BedwarsItemAddon](https://github.com/TheRamU/BedwarsItemAddon)，仅做小部分修改。

> 前置依赖/Depends：
- [BedwarsRel 1.3.6](https://www.spigotmc.org/resources/6799/)
- [BedwarsScoreBoardAddon](https://github.com/TheRamU/BedwarsScoreBoardAddon/)
