# BedwarsItem

> [!CAUTION]
本项目未经过严格测试，由于赶时间有许多生草写法&改法。

可能会有许多BUG，也没有具体玩过“宠物”。发现了麻烦多反馈，望请海涵:)。

> 这个项目是做什么用的？

用途与[BedwarsScoreBoardAddon](https://github.com/LinMoyuu/BedwarsScoreBoardAddon)一致。

因此，本项目存在的意义是为了复刻《花雨庭》服务器的起床战争游戏体验。

当本插件与配套插件完成时，笔者会释出示例端作为参考。
